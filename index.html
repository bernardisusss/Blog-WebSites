Este é um guia bem básico para iniciantes na

Basicamente

A minha meta é mostrar o essencial para que
você possa entender um pouco e começar a testar
seus sites, com isso quem sabe até vire um programador...
por que não? 🫠

Agora chega de papo e vamo ao que interessa!

O que você primeiro tem que entender, é
que existem diferentes tipos de desenvolvimento
ao total 3, sendos esses:

  • O desenvolvimento para web
  • O desenvolvimento para celular
  • E o desenvolvimento de jogos

Cada tipo de desenvolvimento usará diferentes
linguagens, a princípio, meu foco será mostrar
o essencial e o básico para que você compreenda
o "desenvolvimento para web"

Agora você se pergunta... o que seriam essas linguagens
que você falou?? ou mesmo, o que cada uma representa

Bem, de um modo geral, existem diversos tipos de linguagens
para diferentes finalidades, essas linguagens são as
"ferramentas" que podemos utilizar, no nosso caso
de desenvolvimento da web para construirmos sites
para a internet.

Para criarmos os sites, a gente precisa de um editor de código
pois é com ele que as nossas ferramentas ou
as linguagens vão funcionar.

Agora, para um site sua mente é amiga viu?
você pode fazer absolutamente o que quiser,
talvez um blog pessoal, um site de venda de
vendas de seus produtos, site de agendamento
ou mesmo um site para ver a hora... haha vale tudo!

Para começar o primeiro passo é instalar
um editor de código específico

Existem diversos programas que fazem bem esse papel
mas o mais recomendado é o "vs code"
um excelente editor, é leve e totalmente gratuito e
desenvolvido pela própria microsoft, é seguro!
pode ir sem medo! 😎

Então sabendo que editor usar, vou apresentar as
principais tecnologias para um desenvolvedor da web
